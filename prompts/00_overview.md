# Using HyperFrames slash commands in a composition

This guide explains how to author a puma-info video composition with the
HyperFrames skill bundle, independent of which coding agent you use.

## Prerequisites

- The Group D services are built and running (`make video-up`).
- A composition directory created from the template
  (`make video-new-composition NAME=<id>`), which already contains the
  HyperFrames skill bundle.

## Workflow

1. Open your coding agent with the composition directory
   (`compositions/<id>/`) as its working directory.
2. If your agent supports the skill format, the HyperFrames skills are
   discovered automatically from the bundle in that directory. They cover
   the HyperFrames CLI plus animation libraries (GSAP, anime.js, CSS
   animations, Lottie, three.js, the Web Animations API, Tailwind, and the
   HyperFrames media helpers).
3. Describe the scene you want. The skill guides the agent to write
   `index.html` and any referenced sub-compositions in HTML, CSS and JS.
4. Preview locally with `make video-preview NAME=<id>`
   (served at http://localhost:3001).
5. Render with `make video-render NAME=<id>`, producing `output/<id>.mp4`.
   To attach narration generated by Group C, pass the audio track to the
   render orchestrator.

## Reference

- HyperFrames source (Apache-2.0): https://github.com/heygen-com/hyperframes
