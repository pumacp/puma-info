# SKILL: context

- **Role:** secondary source-of-truth — research / reference / context
- **Mutability:** IMMUTABLE — never modified; a fixed record that informs the primary.
- **Contains:** accumulated research, references and prior material that fed and
  corrected `primary/`. Organize by type (`docs/`, `media/`, `refs/`) and, when a
  project has phases, by optional phase subfolders (`01-…/`, `02-…/`).
- **Purpose / how to use:** read-only **context** for generation and for correcting
  the primary. Agents may read it to ground output; they must **not** modify it.
- **Provenance:** produced during the project's research/earlier phases; feeds the
  primary via the author (outside the pipeline).
